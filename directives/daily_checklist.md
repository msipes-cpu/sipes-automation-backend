# Daily Outreach Checklist: The Rule of 100

**Goal**: 100 Primary Actions per day for 10 days (or until 5 pilots are filled).

## Preparation (Night Before)
- [ ] Identify 50 new prospects for Email (add to `first_five_tracking.csv`).
- [ ] Identify 50 new prospects for LinkedIn (add to `first_five_tracking.csv`).
- [ ] Customize the "First Line" context for each in the CSV.

## Execution (Morning - Eat the Frog)
### Email Block (Audio/Video off, deep work)
- [ ] Run `python execution/outreach_manager.py` to generate drafts.
- [ ] Review drafts in Gmail.
    - Check the custom first line.
    - Check the name/company variables.
- [ ] Send 50 Emails.
- [ ] Update `first_five_tracking.csv` status to 'Sent'.

### LinkedIn Block
- [ ] Open LinkedIn Sales Navigator / LinkedIn.
- [ ] Send 50 Connection Requests + Notes / DMs using the script.
- [ ] Update `first_five_tracking.csv` status to 'Sent'.

## Follow-up (Afternoon)
- [ ] Check inbox for replies.
- [ ] Reply immediately to book calls.
    - **Goal**: Get them on a 15-min "Fit Call".
- [ ] Update `first_five_tracking.csv` status to 'Replied' / 'Booked'.

## Review
- [ ] Tracking sheet updated?
- [ ] Hit 100 touches?
- [ ] Any calls booked?
